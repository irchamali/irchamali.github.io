This book is designed as a systematic and applicable guide for beginners who want to learn and master the basics of programming using the Python language. With a step-by-step approach, from the most basic concepts to practical applications, the book equips readers with knowledge and skills that can be immediately applied in various fields. The first chapters provide an introduction to Python, its history, reasons for choosing Python, and software installation. This is followed by an in-depth discussion of basic syntax, data types, variables, and logical and string operations, which form an important foundation for writing code. Readers are then invited to understand program flow through branching and looping, and process data using data structures such as lists, tuples, sets, and dictionaries. Programming skills are further enhanced through an introduction to functions, modular programming, and file processing techniques. Focusing not only on theory, the book also discusses an introduction to the object-oriented programming (OOP) paradigm, data visualization with matplotlib and seaborn, and the application of Python in automating daily tasks and simple web scraping. Finally, a mini-project chapter is provided that encourages readers to design and develop a simple application, so that all the concepts learned can be tested in a real-life project. This book is suitable for use as teaching material, reference book, or self-study guide, both for students, teachers, lecturers, and anyone who wants to start their journey in the world of programming with Python.
