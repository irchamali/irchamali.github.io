Web application development is increasingly complex and demands high speed and efficiency. PHP, as a server-side scripting language, continues to evolve to meet user needs. To maximize the potential of PHP 8, the use of a web framework such as CodeIgniter 4 is crucial. CodeIgniter 4, as one of the most popular PHP frameworks, is known for its lightweight, fast, and flexible design. This textbook is systematically structured to guide students or readers in learning the steps of web application development. Starting with an introduction to PHP 8 and CodeIgniter 4, readers will then be guided through various important concepts such as data types, variables, operators, branching and looping, and techniques for building applications using Model-View-Controller (MVC), the main foundation of the CodeIgniter framework. In addition, this book also discusses other important topics such as working with Git, database migration, page management, implementing an authentication system with Myth:Auth, and managing data with CRUD (Create, Read, Update, Delete) techniques. This textbook not only provides theory but also includes hands-on exercises, code examples, and case studies, which are expected to help readers understand and apply the knowledge they learn. Each chapter is designed to explain concepts simply yet comprehensively, making it suitable for students, novice web developers, and those who want to learn more about the CodeIgniter 4 framework.
